# Agenda

Aplicativo feito englobando os seguintes cursos na Alura Cursos Online:

- [Curso Android I: Crie sua App fantástica com Android Studio][curso1]
- [Curso Android II: Integração com apps e recursos do device][curso2]
- [Curso Android III: Aprimore sua app com Fragments, Google Maps e GPS][curso3]
- [Curso Android com Web Service parte 1: Sincronize sua app com o servidor][curso4]
- [Curso Android com Web Service parte 2: mais técnicas de sincronização][curso5]

O servidor usado no aplicativo pode ser baixado [aqui][server]. Basta rodar o servidor com o o comando `java -jar agenda-server.jar`

*O servidor foi fornecido pela Alura durante as aulas.*
